## 2-以字计量

## 3-取双列

opencc 语法更严格了，仅取出一份【单映射无重表】作为拆分显示辅助，主要用在手机同文输入法软件上。

基于「有歧义拆分」和「无歧义拆分」，二分法取出一个明细，供后面处理

## 4-重复项整合

汉字 +Tab+［多歧义拆分串联 + GBK］
